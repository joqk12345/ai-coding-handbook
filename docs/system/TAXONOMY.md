# Taxonomy Governance

Taxonomy configuration lives in `meta/taxonomy.yaml`.

It controls:
- valid content types
- allowed high-level topics
- canonical concept labels
- publication statuses

Changes to taxonomy require review because they affect linter validity and generated navigation.
