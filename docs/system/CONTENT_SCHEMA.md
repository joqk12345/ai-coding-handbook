# Content Schema

All markdown articles must include frontmatter fields defined in `meta/content-schema.json`.

Required keys:
- `id`
- `title`
- `slug`
- `date`
- `type`
- `topics`
- `concepts`
- `tools`
- `architecture_layer`
- `timeline_era`
- `related`
- `references`
- `status`
- `display_order`

Article body content is never generated and remains the source of truth.
