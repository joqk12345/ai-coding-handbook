# Agent 本质

1. 回归 Agent 本质 - 规划、记忆、工具调用  
   以规划（Planning）、记忆（Memory）、工具调用（Tool Use）为核心能力，构建可执行、可持续优化的智能体。

2. 加速迭代效率 - Headless Mode、Agent SDK、评估生态  
   通过 Headless Mode、Agent SDK 与评估生态（Eval）打通开发-测试-上线闭环，把试错成本降到最低。

3. 重新定义生产关系 - 技术选型简单、AI 迭代 AI  
   让技术选型更轻、更快，形成“AI 迭代 AI”的新范式，人类负责目标与约束，AI 负责实现与优化。
